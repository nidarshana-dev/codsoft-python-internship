# CodSoft Python Programming Internship

This repository contains all the tasks completed for the **CodSoft Python Programming Internship - September 2026 Batch**.

## ✅ Task 1: 1CR Salary Calculator

**Description:** 
A Python program that calculates annual salary from monthly input and adds a 20% performance bonus. If the total yearly income crosses ₹1,00,00,000, it displays a special success message.

**Concepts Used:**
- Variables & Data Types
- User Input / Output
- Conditional Statements (if-elif-else)
- Arithmetic Operations
- f-String Formatting

**How to Run:**
```bash
python calculator.py
```

**Sample Output:**
```
Enter monthly salary: 50000
Your annual salary: Rs600000
Your bonus: Rs120000.0
Total 1 year income: Rs720000.0
Innum Rs9280000.0 venum. Next year pakkalam
```

### ✅ Task 2: Password Generator

**Description:** A Python program that generates a strong random password based on user-specified length. Uses letters, numbers, and symbols.

**Features:**
- User can set custom password length
- Mixes uppercase, lowercase, numbers, and symbols
- Validates minimum length for security

**How to Run:**
1. Run `python password_generator.py`
2. Enter desired password length
3. Get your secure password

**Sample Output:**

  ```
Keep it safe! Don't share with anyone.
```
